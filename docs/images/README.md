# Screenshots Directory

Place your app screenshots here for the README documentation.

## Required Screenshots

1. **banner.png** - Header banner for README (1200x400 recommended)
2. **auction-view.png** - Live auction presenter view
3. **dashboard.png** - Admin dashboard overview
4. **teams.png** - Teams management page
5. **player-pool.png** - Player pool with tier badges
6. **auction-control.png** - Admin auction control panel

## Recommended Dimensions

- Banner: 1200 x 400 px
- Screenshots: 1280 x 720 px (16:9)
- Use PNG format for best quality

## Tips for Good Screenshots

1. Use sample data that looks realistic
2. Show the app in its best state (no errors, loading complete)
3. Highlight key features in each screenshot
4. Consider adding annotations or callouts
5. Ensure text is readable at reduced sizes
